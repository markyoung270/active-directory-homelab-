# Active Directory Home Lab (Windows Server 2022)

## Overview
This project demonstrates hands-on experience building and supporting a Windows Active Directory environment in a home lab.
The lab focuses on core IT support and help desk tasks commonly performed in enterprise environments, including user management, domain authentication, and account troubleshooting.

## Environment
- Windows Server 2022 (Domain Controller)
- Windows 11 Pro (Domain-joined client)
- Hyper-V virtualization
- Active Directory Domain Services (AD DS)
- DNS

## What This Project Covers
- Domain controller installation and configuration
- DNS configuration for Active Directory
- Organizational Unit (OU) design
- Domain user and security group creation
- Windows client domain join
- Domain authentication verification
- Password reset and account unlock workflow
- Troubleshooting DNS, IP addressing, and authentication issues

## Key Skills Demonstrated
- Active Directory Users and Computers (ADUC)
- Windows domain authentication
- User and group management
